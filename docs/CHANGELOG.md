# EduFlow 更新日志

## [Unreleased] - 2025-12-18

### 新增 (Added)
- **目标倒计时功能**：
    - 在 Header 区域增加了目标设置入口（Dialog）。
    - 显示距离目标日期的剩余天数。
    - 采用青色（Cyan）主题，避免红色带来的视觉焦虑。
- **日历条 (Date Strip)**：
    - 实现了当月剩余日期的横向滚动视图。
    - 周末（周六、周日）日期和星期显示为红色。
    - 今日日期高亮显示。
- **背景纹理**：
    - 主内容区域增加了 30px * 30px 的虚线格子背景，提升质感。
- **拖拽排序**：
    - 引入 `@dnd-kit` 实现卡片的网格拖拽排序。
- **单卡刷新**：
    -用于卡片右上角的刷新按钮，可单独重新生成该科目的知识点。

### 修改 (Changed)
- **内容适配逻辑**：
    - 后端 `KnowledgeService` 升级，根据用户 Phase（小学/初中）区分使用不同的知识库，修复了小学用户收到不适宜内容的问题。
- **日历条样式优化**：
    - 移除了日期格子的外边框（Border）。
    - 缩小了字号（星期用细体），使界面更精致。
- **卡片样式**：
    - 内容左对齐。
    - 标题和正文通过冒号（中文或英文）自动分割换行。
- **用户切换体验**：
    - 切换用户时，立即清空当前卡片和目标状态，强制界面刷新，提供明确的视觉反馈。

### 修复 (Fixed)
- **构建错误**：
    - 修复了 `app/page.tsx` 中因代码合并导致的 Parsing Error (重复函数定义、缺失闭合标签)。
- **API 路径错误**：
    - 修复了 `fetch` 调用 URL 中多余空格导致的 404 错误。
